# Proyek Permasalahan Perusahaan Edutech

## Persiapan

Setup environment (python):

1. Membuat Virtual Environment (Windows)

    ``` python
    python -m venv venv
    ```
    ``` bash
    .\venv\Scripts\activate
    ```

2. Install Library

    ``` python
    pip install -r requirements.txt
    ```

3. Jalankan Streamlit

    ``` bash
    streamlit run app.py
    ```

# Edutech Company Problem Project 🚀

## Preparation 🛠️

Setup environment (Python):

1. **Creating Virtual Environment (Windows)** 🖥️

    ``` python
    python -m venv venv
    ```
    ``` bash
    .\venv\Scripts\activate
    ```

2. **Install Libraries** 📚

    ``` python
    pip install -r requirements.txt
    ```

3. **Run Streamlit** 🌐

    ``` bash
    streamlit run app.py
    ```
